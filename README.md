# ReimbursementSystemBackEnd

The front end and automated tests for this application are contained in two separate repositories: ReimbursementSystemFrontEnd and ReimbursementSystemAutomation.

# Overview
- A full stack web application that allows Employees submit reimbursement requests for job-related expenses, which Managers can approve or deny

# Technologies
- Python
- Flask
- JavaScript
- HTML and CSS
- PostgreSQL
- Selenium

# Features
- Employees can submit a reimbursement request
- Managers can approve or deny requests
- Managers can view all requests and a statistics page
